# Python Lab Solutions

This repository contains solutions to a series of Python programming labs.  
Each lab introduces fundamental programming concepts, problem-solving techniques, and practical exercises to strengthen Python skills.

---

## 📂 Repository Structure

- **Lab 1**: Variables, arithmetic operations, input/output, and basic problem-solving
- **Lab 2**: Conditionals, loops, calculator program, daily wages, loyalty bonus, FizzBuzz, PIN attempts
- **Lab 3**: Functions (triangle, greatest number, sum series, factorial, palindrome)
- **Lab 4**: Lists and strings (add/remove items, longest string, duplicate word detection)
- **Lab 5**: Data structures (lists, dictionaries, student grades, shopping cart, employee database, word frequency, inventory management, anagram checker, two-sum problem)
- **Lab 6 Part 1**: System monitoring simulations (login attempts, log analyzer, disk usage, command execution, CPU monitoring)
- **Lab 6 Part 2**: Advanced exercises (pangram detector, find-and-replace, coordinate directions)
- **Lab 7**: File handling and JSON  
  - Reading files with exception handling  
  - Writing user input to files with permission checks  
  - Saving and loading data in JSON format  
  - Validating user input (name, age, email)  
  - Menu-driven program combining all tasks
- **Lab 8**: Task Manager System  
  - Create, complete, and list tasks  
  - Track pending vs completed tasks  
  - Manage task priorities (1–10)  
  - Sort tasks by priority  
  - Bonus: prevent duplicate tasks and count completed vs pending
